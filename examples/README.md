# Examples

This folder contains example data and projects for testing the SecInterp plugin.

## Study Area

The sample data covers a portion of the **LOMA ALTA F14A59** topographic sheet from INEGI (Instituto Nacional de Estadística y Geografía), Mexico.

## Contents

- **sample_data/**: Sample geological and topographic data for testing
- **tutorials/**: Step-by-step tutorial projects (coming soon)

## Data Sources

### Digital Elevation Model (DEM)
The DEM data can be obtained from INEGI's Continental Relief dataset:
- **URL**: https://www.inegi.org.mx/temas/relieve/continental/
- **Sheet**: F14A59 (Loma Alta)

### Geological and Structural Data
The geological polygons and structural measurements are **fictitious data** created solely for demonstration and testing purposes. They do not represent real geological conditions.

## Usage

1. Load the sample QGIS project or add the layers manually
2. Select the DEM raster (`mdearea.tif`)
3. Select a section line (`secc1.shp`)
4. Optionally add geological units and structural data
5. Generate your cross-section profile
